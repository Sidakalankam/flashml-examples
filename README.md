# FlashML Examples

Runnable notebooks for learning FlashML end to end.

## End-to-end Colab quickstart

Open the notebook in Colab:

https://colab.research.google.com/github/Sidakalankam/flashml-examples/blob/main/flashml-end-to-end.ipynb

The notebook trains a tiny text classifier, exports it to ONNX, writes the FlashML bundle files, uploads the bundle, and calls the hosted inference endpoint.
